## ✅ How To
📅 2025.05.20 ~

1. 매주 각자 1질문 가져오기 (+ **꼬리질문 1~2개**)
3. 매주 화요일 ``22:00``
4. 질문 주제에 맞는 파일에 질문 추가
   - React.md
     - Q. 리액트의 useMemo는 무엇인가요?
5. 질문에 대해 토의한 후 정리해서 정답 작성
   - React.md
     - Q. 리액트의 useMemo는 무엇인가요?
     - A. 리액트의 메모이제이션 훅입니다.
